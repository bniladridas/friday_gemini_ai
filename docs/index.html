Friday Gemini AI

Ruby Gem Documentation

Ruby Gem v0.1.4

Overview

Friday Gemini AI is a Ruby gem that provides a simple interface for interacting with Google's Gemini AI models. It allows you to generate text, chat with the model, and process images with text prompts.

Features

  • Simple, intuitive API for Gemini AI models
  • Support for text generation
  • Support for chat conversations
  • Support for image-to-text generation
  • Configurable parameters (temperature, max tokens, etc.)
  • Error handling and logging

Installation

Add this line to your application's Gemfile:

gem 'friday_gemini_ai'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install friday_gemini_ai

Usage

Setup

First, you need to obtain an API key from Google AI Studio. Then, you can initialize the client:

require 'gemini_ai'

# Initialize with API key
client = GeminiAI::Client.new('your_api_key')

# Or use environment variable
# export GEMINI_API_KEY=your_api_key
client = GeminiAI::Client.new

Text Generation

Generate text with a prompt:

response = client.generate_text('Tell me a short joke about programming')
puts response

With additional parameters:

response = client.generate_text(
  'Explain quantum computing in simple terms',
  temperature: 0.3,
  max_tokens: 500
)
puts response

Chat

Have a conversation with the model:

messages = [
  { role: 'user', content: 'Hello, how are you?' },
  { role: 'model', content: 'I\'m doing well, thank you! How can I help you today?' },
  { role: 'user', content: 'Can you explain what machine learning is?' }
]

response = client.chat(messages)
puts response

Image-to-Text

Process an image with a text prompt:

require 'base64'

# Load and encode image
image_path = 'path/to/your/image.jpg'
image_data = File.read(image_path)
base64_image = Base64.strict_encode64(image_data)

# Generate text based on image
response = client.generate_image_text(
  base64_image,
  'Describe what you see in this image'
)
puts response

API Reference

initialize(api_key = nil, model: :pro)

Creates a new Gemini AI client.

api_key String (optional)
Your Google AI API key. If not provided, it will look for the GEMINI_API_KEY environment variable.
model Symbol (default: :pro)
The Gemini model to use. Options: :pro, :flash, :flash_lite
generate_text(prompt, options = {})

Generates text based on a prompt.

prompt String
The text prompt to generate from.
options Hash (optional)
Additional parameters:
  • temperature (Float, default: 0.7): Controls randomness. Lower values are more deterministic.
  • max_tokens (Integer, default: 1024): Maximum number of tokens to generate.
  • top_p (Float, default: 0.9): Nucleus sampling parameter.
  • top_k (Integer, default: 40): Top-k sampling parameter.
chat(messages, options = {})

Has a conversation with the model.

messages Array
An array of message objects, each with 'role' (either 'user' or 'model') and 'content' (the message text).
options Hash (optional)
Same options as generate_text.
generate_image_text(image_base64, prompt, options = {})

Generates text based on an image and a prompt.

image_base64 String
Base64-encoded image data.
prompt String
Text prompt to accompany the image.
options Hash (optional)
Same options as generate_text.

Error Handling

The gem uses the GeminiAI::Error class for all errors. You can catch these errors to handle them gracefully:

begin
  response = client.generate_text('Your prompt')
  puts response
rescue GeminiAI::Error => e
  puts "Error: #{e.message}"
end

Common errors include:

  • Missing or invalid API key
  • Empty prompt
  • API request failures
  • Rate limiting or quota exceeded
