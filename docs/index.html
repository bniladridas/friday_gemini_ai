Introduction

Friday Gemini AI is a lightweight Ruby gem (7.5 KB) that provides a simple interface to interact with Google's Gemini AI models, making text generation easy for Ruby developers.

Documentation

Installation

gem install friday_gemini_ai

Or add to your Gemfile:

gem 'friday_gemini_ai'

Configuration

Set your API key using environment variables:

# In your .env file
GEMINI_API_KEY=your_api_key_here

Or configure programmatically:

require 'gemini_ai'

GeminiAI.configure do |config|
  config.api_key = 'your_api_key_here'
  config.default_model = 'gemini-pro' # Optional
end

Basic Usage

require 'gemini_ai'

# Initialize client
client = GeminiAI::Client.new

# Generate text
response = client.generate_text('Explain quantum computing in simple terms')
puts response

# Chat conversation
messages = [
  { role: 'user', content: 'Hello, how are you?' },
  { role: 'model', content: 'I\'m doing well, thank you! How can I help you today?' },
  { role: 'user', content: 'What is Ruby on Rails?' }
]

response = client.chat(messages)
puts response

API Reference

generate_text(prompt, options = {})

Generates text based on a prompt.

  • prompt (String): The text prompt
  • options (Hash): Optional parameters
    • temperature (Float): Controls randomness (0.0-1.0)
    • max_tokens (Integer): Maximum tokens to generate
    • top_k (Integer): Limits vocabulary to top K options
    • top_p (Float): Nucleus sampling parameter

chat(messages, options = {})

Conducts a chat conversation with the model.

  • messages (Array): Array of message hashes with 'role' and 'content'
  • options (Hash): Same options as generate_text

generate_image_text(image_base64, prompt, options = {})

Generates text based on an image and a prompt.

  • image_base64 (String): Base64-encoded image data
  • prompt (String): Text prompt about the image
  • options (Hash): Same options as generate_text

Real-World Examples

Creative Writing

Generating a haiku about Ruby programming:

client.generate_text('Write a haiku about Ruby programming')

Gems sparkle bright red,
Code flows with elegant ease,
Joy in every line.

Technical Information

Having a conversation about Ruby frameworks:

messages = [
  { role: 'user', content: 'Hello, can you tell me what Ruby is?' },
  { role: 'model', content: 'Ruby is a dynamic, object-oriented programming language known for its simplicity and productivity.' },
  { role: 'user', content: 'What are some popular Ruby frameworks?' }
]

client.chat(messages)

Here are some popular Ruby frameworks:

Web Frameworks: Ruby on Rails (most popular), Sinatra (lightweight), Hanami (modular)

API Frameworks: Grape, Rails API mode

Testing: RSpec, Minitest, Cucumber

Background Jobs: Sidekiq, Resque

Customizing Parameters

Getting a concise definition with lower temperature:

client.generate_text('Explain what an API is in one sentence', temperature: 0.3, max_tokens: 50)

An API (Application Programming Interface) is a set of rules and specifications that allows different software applications to communicate and exchange data with each other.
